# Security

This project is a **frontend demonstration**. It does **not** process real payments or store orders unless you connect your own backend and secrets.

## Reporting issues

If you find a security problem in this repository (dependency vulnerability, unsafe pattern, etc.), please open a **private security advisory** on GitHub if available, or email the maintainer with details so it can be addressed responsibly.

Do not post exploit details in public issues before a fix is agreed.

## Supply chain

Run `npm audit` periodically and upgrade dependencies through normal PRs.
